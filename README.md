# Light Puzzle Game - Головоломка с лампочками

Консольная игра-головоломка на C#, в которой нужно зажечь все лампочки на игровом поле, перемещая луч света от текущей позиции.

## Описание игры

Игровое поле представляет собой сетку 6x6, на которой расположены:
- **Стены (#)** - препятствия, через которые не проходит свет
- **Негорящие лампочки (O)** - их нужно зажечь
- **Горящие лампочки (☼)** - уже зажженные лампочки
- **Текущая позиция (@)** - место, откуда начинается луч света

Игрок начинает с одной зажженной лампочки и должен последовательно зажигать остальные, направляя луч света в разные стороны.

## Управление

- **W** - зажечь лампочки вверх от текущей позиции
- **S** - зажечь лампочки вниз от текущей позиции
- **A** - зажечь лампочки влево от текущей позиции  
- **D** - зажечь лампочки вправо от текущей позиции
- **R** - сбросить игру и начать заново
- **Q** - выйти из игры

## Правила игры

1. Луч света движется в выбранном направлении до первой встреченной стены или уже горящей лампочки
2. Все лампочки на пути луча зажигаются
3. После каждого хода текущая позиция перемещается на последнюю зажженную лампочку
4. Цель игры - зажечь все лампочки на поле
5. Игра заканчивается победой при зажжении всех лампочек или тупиком, если нет доступных ходов

## Игровое поле по умолчанию
- **#** - стена
- **O** - негорящая лампочка
- **@** - Начальная позиция

## Особенности реализации

- Консольный интерфейс
- Проверка победы после каждого хода
- Проверка тупиковых ситуаций
- Возможность сброса игры
- Отображение текущего статуса (количество зажженных лампочек, позиция)

## Требования для запуска

- .NET SDK 8.0 или выше
- Любая операционная система, поддерживающая .NET 8.0

## Сборка и запуск

```bash
# Клонировать репозиторий
git clone https://github.com/yourusername/light-puzzle-game.git

# Перейти в директорию проекта
cd light-puzzle-game

# Собрать проект
dotnet build

# Запустить игру
dotnet run
Структура проекта
Program.cs - основной код игры с классами Program и LightPuzzleGame

LightPuzzleGame.csproj - файл конфигурации проекта

Как играть
1)Запустите игру командой dotnet run
2)Используйте клавиши WASD для направления луча света
3)Старайтесь спланировать последовательность ходов так, чтобы зажечь все лампочки
4)При возникновении тупика используйте R для перезапуска
